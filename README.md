# Regression-Based Business Insights & Model Interpretation

## Business Problem Summary

The objective is to identify factors associated with monthly sales across retail stores using regression analysis.

## Dataset Description

The dataset contains information related to store characteristics, marketing activities, customer behaviour, and sales performance.

## Dependent Variable

- monthly_sales

## Independent Variables

- marketing_spend
- footfall
- avg_discount_pct
- inventory_availability_pct
- customer_rating
- region
- store_type

## Regression Approach

Two simple regression models and one multiple regression model were developed.

## Dummy Variable Approach

Categorical variables were transformed using dummy variables while keeping one category as the reference category.

## Final Model Selected

Multiple regression model.

## Business Recommendation

Focus on marketing efficiency, inventory management, and customer satisfaction.

## Assumptions and Limitations

- Association does not imply causation.
- External factors may influence sales.
