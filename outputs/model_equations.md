# Model Equations

## Simple Regression Model 1

monthly_sales = β0 + β1(marketing_spend)

Higher marketing spend is associated with higher monthly sales.

## Simple Regression Model 2

monthly_sales = β0 + β1(footfall)

Higher customer footfall is associated with higher monthly sales.

## Multiple Regression Model

monthly_sales = β0 + β1(marketing_spend) + β2(footfall) + β3(inventory_availability_pct) + β4(customer_rating) + Dummy Variables

## Dummy Variables

- Region and store type were converted into dummy variables.
- One category was omitted and used as the reference category.
- This avoids the dummy variable trap.

## Final Model Selected

The multiple regression model was selected because it considers multiple business factors simultaneously and provides higher explanatory power than simple regression models.
